# AI Email Classification System

## Overview
This project is an AI-powered email classification system that analyzes email content and categorizes it based on intent, sentiment, urgency, and risk.

## Features
- Email intent classification
- Sentiment analysis
- Urgency detection
- Risk assessment
- Clean UI for analysis

## Tech Stack
- Frontend: React (TypeScript)
- Backend: FastAPI
- ML: Pandas, NumPy
